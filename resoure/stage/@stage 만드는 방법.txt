1. window.exe 파일 실행

2. 입력창 클릭해서 이미지 선택
-> 이미지 선택시 아래쪽에 해당 이미지 나옴

3. 빈 블록 만들 색 지정(RGB)
처음 실행시 1번에 체크 되어있고 255 255 255 적혀있음
숫자는 각각 R G B 색을 가리킴
255 255 255 는 흰색
0 0 0 은 검은색

각 번호 옆에 R G B 색을 지정하고 체크를 하면
텍스트파일로 만들어 질 때 해당 색은 블럭이 아니라 빈 공간으로 만들어버림

체크만 해두고 숫자 지정안해두면 작동 안함

4. 블럭 사이즈 지정
-> 처음 실행시 40 40 적혀있음
사실 블럭 사이즈가 아니라 블럭의 갯수
30 50 입력하면
가로 30개, 세로 50개 총 150개의 블럭을 만듦

5. 텍스트파일로 버튼
클릭하면 그대로 텍스트파일로 만듦

6. stage 텍스트파일
stage_0.txt 처럼 stage_ 뒤에 숫자가 들어가야 함

stage_0 파일이 있으면 stage_0 부터 시작
없으면 stage_1 부터 시작 함

stage_1 부터 stage_5 까지 있을건데
stage_0 만들어도 되고, stage_6부터 쭉 이어서 만들어도 상관 없음
이미 만들어져 있는 파일 수정해도 상관 없음

스테이지 파일에 빈공간은 '' 라고 표시되어 있긴 한데 아무 표시 없어도 잘 작동 함
